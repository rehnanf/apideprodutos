4.1 API Resources no Laravel
Explique:
Qual é o objetivo de utilizar API Resources?

O objetivo dos API Resources no Laravel é transformar e padronizar os dados retornados pela API em formato JSON.

Em quais situações eles são úteis no desenvolvimento de APIs?

Os API Resources são úteis principalmente quando:

- É necessário padronizar as respostas da API
- Queremos esconder campos sensíveis (ex: senhas, dados internos)
- Precisamos formatar dados (ex: datas, valores monetários)
- Trabalhamos com relacionamentos entre modelos (ex: produto com categoria)
- A API precisa ser consumida por frontend (React, Vue, Mobile)

4.3 Testes Automatizados no Laravel
Responda às seguintes perguntas:

1. Para que servem testes automatizados em uma aplicação Laravel?

Testes automatizados em uma aplicação Laravel servem para garantir a qualidade, estabilidade e confiabilidade do software,
permitindo que funcionalidades sejam verificadas automaticamente sem a necessidade de testes manuais repetitivos.
